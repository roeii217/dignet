# DigitalNet — שיווק דיגיטלי מתקדם ואסטרטגיה

אתר דמו מקצועי לסוכנות שיווק דיגיטלי.

## קבצים

| קובץ | תיאור |
|------|-------|
| `index.html` | האתר המלא — HTML/CSS/JS קובץ יחיד |
| `digitalnet_logo.png` | לוגו החברה |

## העלאה ל-Vercel

1. דחוף את הקבצים ל-GitHub repo
2. היכנס ל-[vercel.com](https://vercel.com)
3. לחץ **Add New → Project**
4. בחר את ה-repo מ-GitHub
5. לחץ **Deploy** — הכל אוטומטי ✓

## טכנולוגיות

- HTML5 / CSS3 / Vanilla JS
- Google Fonts (Heebo + Montserrat)
- ללא תלויות חיצוניות — עובד מיד
